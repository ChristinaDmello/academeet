# 🎓 Academeet

Academeet is a college community platform designed for students and faculty to communicate, collaborate, and stay updated with campus activities.

## ✨ Features

### 👨‍🎓 Student Features
- View and download notes
- View announcements
- Register for events
- Post and view doubts
- Access Lost & Found items

### 👩‍🏫 Faculty Features
- Upload and manage notes
- Create, edit, and delete announcements
- Create and manage events
- Answer student doubts
- Access Lost & Found items

## 🛠 Technologies Used

### Frontend
- React
- Vite
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
